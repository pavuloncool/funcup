# ADR-007: i18n with Manual Translation Files

**Date:** 2026-04-08  
**Status:** Accepted  
**Feature:** 002-qr-coffee-platform

## Context
Need internationalization support for the multi-language MVP (EN + PL).

## Decision
Create @funcup/i18n package with:
- JSON-like translation files: en.ts, pl.ts
- Functions: setLocale(), getLocale(), t(key, params)
- Keys: common, auth, coffee, roaster, hub, profile, errors

## Consequences
- ✅ Simple, no external dependencies
- ✅ Type-safe translations
- ⚠️ Manual translation management (acceptable for MVP)
