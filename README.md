# funcup - the First Structured Feedback Loop 4 Coffee Roasters and Drinkers

## Project Overview

funcup is a platform connecting coffee roasters and drinkers through QR codes and structured feedback loops.

### Key Features
- QR code scanning for coffee products
- Coffee logging and tracking
- Roaster profiles and verified systems
- Coffee Hub discovery feed
- Progressive sensory reputation system
